1 - Objetivo 

Garantir que o e-commerce Livelo funcione corretamente em todas as funcionalidades principais e secundárias, assegurando uma experiência de compra satisfatória para os usuários. 

  

O plano cobre testes funcionais. 

  

2 - Resumo 

O resumo deste plano de teste é assegurar que o site da Livelo funcione corretamente e ofereça uma experiência de usuário satisfatória. O teste irá atualizar a validação das funcionalidade do site para garantir que ele atenda aos requisitos estabelecidos e proporcione uma experiência de compra eficiente e segura. 

  

O objetivo desses testes são, realizados para que funcione bem todos os componentes do site e aplicativo, para que o usuário tenha a melhor experiencia na usabilidade e tenha vantagens, como nenhum outro site tem, nossa missão é ter uma boa relação clientes e parceiros. 

Os testes serão feitos exclusivamente em desktop e smartphone Iphone.

7 - Critérios usados 

  

Nível Crítico 

  

Cadastro 

Para critério de aceite na Feature de cadastro é assegura que a funcionalidade de cadastro atenda a todos os requisitos e esteja funcionando corretamente em todos os aspectos. 

  

Login 

Para critério de aceite na Feature de login é garantir que o sistema de login funcione corretamente, permitindo acesso para credenciais corretas e bloqueando tentativas inválidas de forma segura. 

  

Pagamento 

Para critério de aceite na Feature de pagamento é assegura que a funcionalidade de pagamento complete as transações de forma correta, segura e com confirmação para o usuário. 

  

Entrega 

Para critério de aceite na Feature de entrega é assegura que a funcionalidade de entrega funcione corretamente, completando o processo de entrega e mantendo o usuário informado. 

  

Nível Alto 

  

Carrinho 

Para critério de aceite na Feature de carrinho é garantir que todas as operações básicas do carrinho (adicionar, atualizar, remover e visualizar o total) estejam operando conforme esperado. 

  

Pesquisa/Busca 

Para critério de aceite na Feature de pesquisa/busca é Relevante, assegurar que os resultados da busca devem ser úteis e correspondente às palavras ou termos que o usuário procurou. 

  

Suporte ao cliente 

Para critério de aceite na Feature de suporte ao cliente é Acessível, suporte ao cliente deve ser fácil de encontrar e usar, garantindo que todos os usuários possam obter ajuda de maneira simples e eficiente. 

  

Categorias 

Para critério de aceite na Feature de categorias é garantir que todas as categorias estejam visíveis e acessíveis para os usuários. 

  

Nível Médio 

  

Carrossel 

Para critério de aceite na Feature de carrossel é Funcional, que é garantir que ele funcione corretamente—ou seja, carregue, navegue e exiba conteúdo. 

  

Compra pontos 

Para critério de aceite na Feature de compra pontos é Funcional, que é garantir que o processo de compra seja simples e intuitivo, os dados do usuário devem ser protegidos e o usuário deve receber confirmação da compra. 

  

Juntar pontos 

Para critério de aceite na Feature de juntar pontos é Funcional, que é garantir que a opção de juntar pontos está visível e operante, o processo de junção deve ser simples e intuitivo, os dados do usuário devem ser protegidos e deve receber a confirmação da junção. 

  

Transferir pontos 

Para critério de aceite na Feature de Transferir pontos é Funcional, a opção de transferir pontos deve estar visível e operante, o processo de transferência deve ser simples e intuitivo e o usuário deve receber confirmação da transferência. 

  

Usar pontos 

Para critério de aceite na Feature de Usar pontos é Funcional, a opção de usar pontos deve estar visível e operante, o processo de uso deve ser simples e intuitivo e o usuário deve receber confirmação do uso dos pontos. 

  

Clube livelo 

Para critério de aceite na Feature de Clube livelo é Funcional, a seção do Clube Livelo deve estar visível e operante, os benefícios do clube devem ser claramente apresentados e aplicados e o usuário deve receber confirmação da adesão e detalhes dos benefícios. 

  

Cliente Clube livelo 

Para critério de aceite na Feature de Cliente Clube livelo é Funcional, a seção de Cliente Clube Livelo deve estar visível e operante, a navegação deve ser intuitiva e fácil de usar os dados devem ser protegidos e o usuário deve receber confirmação de qualquer alteração na conta. 

  

  

Nível Baixo 

  

Conheça livelo 

Para critério de aceite na Feature de Conheça Livelo é Funcional, Informar todas as informações do site, Informação de como utilizar os pontos. 

  

Aplicativo 

Para critério de aceite na Feature de Aplicativo é Funcional, Aceitação de funcionamento em todos os Smartphones e banner links coerentes, e eficaz. 

  

Descontos no site 

Para critério de aceite na Feature de Descontos no site é Funcional, Cliente deve ter descontos em sites em parceiros, por meio de cupom e realização de compras em quantidade, também ter descontos no Site por ser do Clube LIVELO. 

  

Para empresas 

Para critério de aceite na Feature de Para empresas é Funcional, Devem ser considerados contratos de fidelidade, Pontos de Incentivo e Recompensas por utilizar o aplicativo e Site. 

  

Institucional 

Para critério de aceite na Feature de Institucional é Funcional, Passar todas as informações possíveis pro cliente, como Norma de Segurança, trabalhe conosco, Conduta, e Testes de Links. 

  

Redes sociais 

Para critério de aceite na Feature de Redes sociais é Funcional, Iremos mostrar todas as novas fotos, e promoções por meio das redes sociais, ter direcionamento de Tela para as páginas nossas. 

  

Parceiros 

Para critério de aceite na Feature de Parceiros é Funcional, Direcionamento para página de parceiros, Logos e Cores dos parceiros dentro do SITE/APP. 

  

Dúvidas 

Para critério de aceite na Feature de Dúvidas é Funcional, Expor todas as Perguntas e Respostas frequentes no site, Ter aba de pesquisa para os clientes. 

  

IA livelo 

Para critério de aceite na Feature de IA Livelo é Funcional, Ter conversa com Robo antes de passar pros funcionários, o cliente ter todas as suas perguntas respondidas pelo ChatBot da IA. 

  

Destaques para você 

Para critério de aceite na Feature de Destaques para você é Funcional, Iremos analisar o perfil de cada cliente para mostrar promoções, colocar em destaque parceiros faz o perfil desse cliente, e validação de teste de links entre cliente/ Parceiro. 

  

Ofertas de seleção de cliente 

Para critério de aceite na Feature de Ofertas de seleção de cliente é Funcional, Deve Obter links para os parceiros, cronometro em novas ofertas, compras e pontuação para os clientes.
